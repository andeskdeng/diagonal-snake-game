# Diagonal Snake Game 🐍

A modern version of the classic Snake game built as a programming project to practice game logic, object movement, collision detection, and user interaction.

## Project Overview

This project recreates the classic Snake game where players control a growing snake, collect food, and avoid collisions.

The project was built to explore:

- Game development fundamentals
- Object-oriented programming concepts
- Collision detection
- Player input handling
- Real-time updates and game loops
- Algorithmic thinking

## Features

✅ Snake movement system  
✅ Food collection mechanics  
✅ Snake growth logic  
✅ Collision detection  
✅ Score tracking  
✅ Keyboard controls  
✅ Game over conditions

## Technologies Used

- Python
- Pygame
- Object-oriented programming
- Basic game algorithms

## Controls

| Key | Action |
|------|---------|
| ↑ | Move Up |
| ↓ | Move Down |
| ← | Move Left |
| → | Move Right |
